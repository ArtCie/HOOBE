# HOOBE
All rentable vehicles(like scooters, cars etc.) in one web application!

## Requirements:
Docker version 20.10.11, build dea9396
+ nginx:1.17.8-alpine
+ php:7.4.3-fpm-alpine3.11

## How to run
```
git clone https://github.com/ArtCie/HOOBE
```
```
cd HOOBE
```
```
docker-compose up
```
